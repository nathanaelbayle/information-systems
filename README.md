# Getting started

- First, clone the repository 
```bash
git clone https://github.com/nathanaelbayle/information-systems
``` 

- Go in the `information-systems` folder and start the docker
```bash
docker-compose up
```

- Then go to your [this url](http://localhost:8069/).
